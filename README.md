# SpriteSmith Studio

Manual puppet-rig editor for browser-based 2D character animation.

## Workflow
1. Upload a sprite.
2. Add parts from the dropdown list.
3. Drag each part's box, resize the crop, and move the pivot dot.
4. Link parts through the Parent dropdown.
5. Pick a pose and bake a sprite sheet.

The project does not rely on auto-slicing. Parts are manually placed so tilted side-scroller characters can be rigged more predictably.
