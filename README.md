# SpriteSmith Studio

A free, GitHub Pages-ready character and sprite designer.

## Features

- Character name and theme
- Clothing selection
- Body proportion sliders
- Hair, eye, facial, and skin customization
- Ability toggles
- Live SVG preview
- Multiple pose previews
- JSON export
- Local browser save via `localStorage`

## Why it is free

This repository is static-only:
- no database
- no paid API
- no server
- no backend required

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder.
3. Enable GitHub Pages from the repository settings.
4. Select the `main` branch and root folder.

## Optional GitHub Actions deploy

A Pages workflow is included in `.github/workflows/pages.yml`.
