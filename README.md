# PuppetSprite Studio

A free, static, browser-based uploaded-image sprite rig and sprite sheet baker.

## What it does

- Upload one image
- Remove the background in-browser
- Animate the uploaded character with separate torso, hair, arms, and legs motion
- Keep the main preview always visible
- Bake a transparent sprite sheet
- Export JSON metadata

## Deployment

Place `index.html` at the root of a GitHub repository and enable GitHub Pages. No backend is required.

## Mobile

This version is a single HTML file, so it is designed to work both on GitHub Pages and when opened directly from a browser on mobile.
