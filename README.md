# Sprite Rigger AI

A web-based tool to transform static images into animated side-scroller sprites.

## Features
- 🖼️ **Smart Upload:** Upload any character image.
- 🗑️ **AI Matting:** Automatically removes backgrounds using `rembg`.
- ⚙️ **Procedural Rigging:** Separates body parts (Head, Torso, Arms, Legs) for manipulation.
- 🎬 **Live Preview:** Generates side-by-side GIFs showing Idle, Walk, and Attack animations.
- 🔄 **Direction Switching:** Logic included for mirroring the sprite.
