# SpriteSmith Studio

A free, static, browser-based character and sprite studio.

## Includes
- image upload
- live canvas preview
- pose cycling
- sprite sheet baking
- PNG export
- JSON export
- local browser save

## Deploy free on GitHub Pages
1. Put `index.html` in the repo root.
2. Commit to GitHub.
3. Enable GitHub Pages on the repository.
4. Open the Pages URL on desktop or mobile.

## Notes
For local testing, use a static server or GitHub Pages instead of opening the HTML file directly from a phone file manager.
