# SpriteSmith Studio

A free, static, browser-based character and sprite studio that is ready for GitHub Pages.

## What it does

- Upload an image and preview it in a real canvas loop
- Design a character with clothing, proportions, features, and abilities
- Cycle through multiple poses and directions
- Bake a downloadable sprite sheet PNG
- Export a JSON project file
- Save locally in the browser with `localStorage`

## What it does not require

- No backend
- No database
- No API key
- No paid service

## Deploy free on GitHub Pages

1. Create a GitHub repository.
2. Put all files from this folder in the repo root.
3. Enable GitHub Pages in repository settings.
4. Select the `main` branch and `/ (root)` folder.

The included `.github/workflows/pages.yml` can publish automatically too.

## Mobile

This app is designed to work in a mobile browser. For local testing, open it through a static local server or GitHub Pages instead of opening the file directly from the file manager.
